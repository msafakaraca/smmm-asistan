# SMMM-AI - AI Motion Video Prompt (Landing Page)

## Video Stili
Modern SaaS tanıtım videosu, koyu-mor/mavi gradient arka plan, neon vurgu renkleri (mavi, mor, turuncu), glassmorphism UI mockup'ları, smooth 60fps animasyonlar, 45-60 saniye.

**Müzik:** Minimal, teknolojik, ilham verici ambient beat (düşük tempo, epic build-up)

---

## SAHNE 1 (0-5 sn) - PROBLEM

**Görsel:** Karanlık bir muhasebe ofisi masası. Üst üste yığılmış kağıt beyannameler, dağınık dosyalar, post-it'ler. Masa üstünde bir bilgisayar ekranında GİB portalı açık, CAPTCHA kodu bekliyor. Bir saat hızla ileri sarılıyor (timelapse efekti). Kırmızı uyarı ikonları beliriyor: "Süre Doldu", "Hata", "Tekrar Deneyin".

**Metin (fade-in):** *"Her ay aynı döngü... 100+ mükellef, 50+ beyanname türü, saatlerce manuel iş."*

---

## SAHNE 2 (5-10 sn) - DÖNÜŞÜM

**Görsel:** Ekran kararıyor. Ortadan parlak mavi bir ışık hüzmesi yayılıyor. SMMM-AI logosu holografik olarak beliriyor. Logo etrafında dijital parçacıklar dönüyor. Arka planda koyu mor-mavi gradient.

**Metin:** **"SMMM-AI"** (büyük, bold, neon glow efekti)
Alt metin: *"Mali Müşavirliğin Dijital Dönüşümü"*

## SAHNE 3 (10-18 sn) - DASHBOARD

**Görsel:** Modern, karanlık temalı dashboard UI'ı 3D perspektifte beliriyor. Glassmorphism kartlar üzerinde animasyonlu sayaçlar yukarı doğru sayıyor: "324 Mükellef", "1,247 Beyanname", "%98.5 Tamamlanma". Pasta grafik animasyonla doluyor. Sağda canlı aktivite akışı scroll ediliyor. Kartlar hafif float efektiyle hareket ediyor.

**Metin overlay:** *"Tüm ofisiniz tek ekranda"*

---

## SAHNE 4 (18-25 sn) - GİB BOT OTOMASYONU

**Görsel:** Ekranın sol tarafında bot terminal penceresi açılıyor, yeşil metin satırları hızla akıyor: "Mükellef: ABC Ltd. ✓", "KDV1 Beyanname bulundu ✓", "PDF indiriliyor... ✓". Sağ tarafta beyanname kontrol tablosu, satırlar tek tek yeşil tik işareti alıyor. Ortada bir robot ikonu/avatar, gözleri mavi parıldıyor, çevresinde dijital veri akışı dönüyor. Progress bar %0'dan %100'e animasyonla doluyor.

**Metin overlay:** *"GİB Bot: 100 mükellefi dakikalar içinde tarar"*

---

## SAHNE 5 (25-32 sn) - KONTROL ÇİZELGESİ

**Görsel:** Büyük bir tablo/spreadsheet görünümü. Satırlarda mükellef isimleri, sütunlarda beyanname türleri (KDV1, BA, BS, Muhtasar, vb.). Hücreler animasyonla renk değiştiriyor: gri → yeşil (verildi), gri → sarı (bekliyor). Bazı hücreler kırmızı yanıp sönüyor (eksik beyanname uyarısı). Kamera tabloyu yukarıdan aşağıya smooth scroll ediyor. Virtual scrolling efektiyle yüzlerce satır görünüyor.

**Metin overlay:** *"Tüm beyannameleriniz tek çizelgede"*

## PROJE ÖZELLİK ÖZETİ (Referans)

### Ana Modüller
- **Dashboard**: İstatistikler, grafikler, aktivite akışı, hızlı işlemler
- **Mükellef Yönetimi**: CRUD, Excel import, toplu işlemler, detay paneli
- **GİB Bot**: Otomatik beyanname tarama, PDF indirme, CAPTCHA çözme
- **Beyanname Takip**: Aylık durum çizelgesi, renk kodlu statüler
- **Kontrol Modülleri**: KDV, KDV2, KDV9015, SGK PDF parse ve takip
- **Dosya Yönetimi**: Supabase Storage, klasör yapısı, PDF viewer
- **Toplu Gönderim**: E-posta, WhatsApp, SMS ile beyanname/tahakkuk gönderimi
- **Duyuru Sistemi**: Şablonlar, zamanlama, çoklu kanal
- **Hatırlatıcılar**: Takvim, görevler, WhatsApp bildirimi
- **Görev Yönetimi**: Kanban, öncelik, atama, yorumlar
- **Takip Çizelgesi**: Dinamik kolonlar, özelleştirilebilir tablo
- **Şifre Kasası**: AES-256 şifreli merkezi şifre yönetimi
- **PDF Araçları**: Birleştir, ayır, sıkıştır, dönüştür, OCR (9 araç)
- **Mail Yönetimi**: Google/Microsoft OAuth, SMTP, gelen/giden kutusu
- **AI Asistan**: Vergi mevzuatı chatbot
- **Ayarlar**: Kullanıcı, ofis, entegrasyon, abonelik yönetimi

### Teknik Öne Çıkanlar
- Multi-tenant SaaS mimarisi
- AES-256-GCM şifreleme
- Row Level Security (RLS)
- Virtual scrolling (100,000+ kayıt)
- WebSocket real-time iletişim
- 35+ API endpoint
- 60+ React component
- 20 veritabanı modeli
